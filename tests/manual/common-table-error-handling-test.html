Test 1: 有効なデータ

正常なデータでテーブルが表示されることを確認します。

基本情報

メールアドレス test@example.com

Test 2: バリデーションエラー

データ形式に問題がある場合のエラー表示を確認します。

エラー

データの形式に問題があります

詳細:
  • 行 0: サポートされていない行タイプ 'invalid_type' です

エラーID: CT_ABC12345_1234567890
この問題が続く場合は、システム管理者にお問い合わせください。

Test 3: バリデーション警告

データに軽微な問題がある場合の警告表示を確認します。

  • 行 0, セル 1: 有効なURL形式ではありません
